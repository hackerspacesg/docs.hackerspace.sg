## Bank Transfers {#bank}

The HackerspaceSG bank account is with **Standard Chartered (Singapore)**.

The account number is:

    5701304090

Other bank account details, if required:
- Bank code: `9496`
- Branch code: `057` (VivoCity)
- SWIFT BIC: `SCBLSG22XXX`

You may set up a recurring payment standing instruction on your bank's internet banking website.

Please refer to instructions at the respective bank websites:

- [DBS/POSB Bank](https://www.dbs.com.sg/personal/support/bank-local-funds-transfer-setup-recurring-funds-transfer.html)
- [UOB Bank](https://www.uob.com.sg/personal/support/tmrw-user-guide/fundstransfer.page#scheduledtransfer) (supported on mobile)
- [OCBC Bank](https://www.ocbc.com/personal-banking/online-banking/funds-transfer.html)
- [Standard Chartered Bank](https://www.sc.com/sg/bank-with-us/transfers-and-payments-guide/) (supported on mobile)
- [HSBC](https://www.hsbc.com.sg/content/dam/hsbc/sg/documents/ways-to-bank/online/local-bank-transfer-guide.pdf) (PDF)
- [Maybank](https://www.maybank2u.com.sg/en/personal/payments/local/fund-transfers/local-funds-transfer.page) (supported on mobile)

As of March 2022, only UOB, Standard Chartered, and Maybank allow creating standing instructions in their mobile apps.

## Credit/Debit Cards {#cards}

### Open Collective

HackerspaceSG uses a platform called Open Collective to manage credit/debit card recurring payments through Stripe.

* [HackerspaceSG's Open Collective page](https://opencollective.com/hackerspacesg)

Stripe's fees are 3.4% + $0.50 per transaction.

### PayPal

HackerspaceSG also supports PayPal.

Enter an amount into the following form field to set it up as a monthly PayPal payment:

PayPal's fees are 3.9% + $0.50 for Singapore-issued cards, and 4.4% + $0.50 for non-Singapore-issued cards.

### GitHub Sponsors

* [HackerspaceSG's GitHub Sponsors page](https://github.com/sponsors/hackerspacesg/)

GitHub Sponsors doesn't charge any fees. As all payments are processed in USD, some currency fluctuation should be expected.
